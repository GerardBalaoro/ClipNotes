Take notes smartly with **ClipNotes**.  
**ClipNotes** is a .NET application that records texts from your clipboard.

#### Product Features

*   Smart Content Aware Clipboard Recording
*   Modern Design
*   Export to Text or RichText Files
*   Supports Richtext Format
